🔌 Integracja z MT4

Skopiuj swój User ID i wklej go w ustawieniach EA w MetaTrader 4. EA automatycznie otworzy panel transakcji przy każdej pozycji.

Twój User ID
Jak skonfigurować EA:
1. Pobierz plik TraderLogJournal_EA.mq4
2. W MT4: File → Open Data Folder → MQL4 → Experts
3. Wklej plik i naciśnij F7 aby skompilować
4. W MT4: Tools → Options → Expert Advisors → Allow WebRequests
5. Dodaj ten URL do listy dozwolonych:
https://ygrkcynyduuflzvbkkvo.supabase.co
6. Przeciągnij EA na wykres → wklej User ID → wklej numer konta MT4
7. Włącz AutoTrading w MT4
⚙️ Numer konta MT4 ustawiasz osobno dla każdego konta w Moje konta → Edytuj konto
